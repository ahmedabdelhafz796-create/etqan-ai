# The Triple Analysis — مصادر الكتاب

هذا المجلد يحتوي على العمل الجاري لكتاب "The Triple Analysis"، وهو مشروع أكبر بكثير من الملف السابق `../كتاب-التحليل-الثلاثي.md` (ذلك الملف كان مسودة أولى أصغر بهيكل مختلف من 20 فصلًا؛ هذا المشروع يحل محله كخط العمل الرئيسي).

## الهيكل

- `00-outline.md` — جدول المحتويات المعتمد (المرجع الوحيد للترقيم والهيكل).
- `PROGRESS.md` — جدول تتبع حالة كل فصل (لم تبدأ / قيد الكتابة / مكتملة).
- `chapters/` — كل فصل في ملف Markdown مستقل، مرقّم بنفس ترقيم `00-outline.md`.
- `figures/` — كل الأشكال التوضيحية كملفات SVG متجهية أصلية، مولّدة بالكامل بواسطة `scripts/generate_figures.py`.
- `scripts/figlib.py` — مكتبة مشتركة لرسم الشموع اليابانية والمربعات والأسهم وبيانات الأسعار الاصطناعية.
- `scripts/generate_figures.py` — سكربت يولّد كل شكل من الأشكال الـ39 في الكتاب (شمعة واحدة لكل دالة).
- `build_pdf.py` — يجمّع كل ملفات `chapters/*.md` (بما فيها مراجع الصور) في HTML ثم PDF نهائي واحد.

## طريقة العمل

الكتاب يُكتب فصلًا فصلًا عبر عدة جلسات، لأن حجمه (33 فصلًا + ملاحق) يجعل كتابته دفعة واحدة غير واقعية بجودة حقيقية. بعد كل فصل يتم تحديث `PROGRESS.md` وحفظ العمل عبر commit.

تُجمَّع ملفات `chapters/*.md` بالترتيب في ملف HTML واحد ثم PDF نهائي (غلاف + فهرس + محتوى + رسوم + مسرد)، باستخدام تنسيق عربي كامل (خط Noto Naskh Arabic، اتجاه RTL).

## الرسوم التوضيحية

كل شكل في الكتاب أصلي 100%: مصمم ومُنتَج برمجيًا عبر matplotlib (`scripts/generate_figures.py`)، وليس منسوخًا أو مستخرجًا من أي صورة مرجعية. أي بيانات سعرية داخل الأشكال اصطناعية (Synthetic) تُنتَج عشوائيًا لمحاكاة سلوك الأسواق فقط، وليست بيانات سوق حقيقية. لإعادة توليد جميع الأشكال بعد أي تعديل:

```bash
cd scripts && python3 generate_figures.py
```

ثم لإعادة بناء PDF الكامل:

```bash
python3 build_pdf.py
```
